# Эталонные транскрипции

Положите сюда файлы `.txt` с эталонными (ground truth) транскрипциями для тестовых видео.

Формат: один файл `.txt` на одно видео, имя файла должно совпадать с именем видео.

Пример:
- `sample_ground_truth.txt` — для `data/raw_videos/sample.mp4`

Файлы с эталонами **не коммитятся** в репозиторий (см. `.gitignore`).
